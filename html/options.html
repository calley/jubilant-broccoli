MacOS users receive different looking notifications due to chrome wanting to respect the MacOS "Do Not Disturb" functionality (among other things too). Disregard this message if you do not want to change this. More info can be found here.
--If you would like to receive the "original" notifications, go to
" chrome://flags/#enable-native-notifications " in a new tab and set "Enable native notifications" to "Disabled". This can be reverted at any time. Enjoy!
